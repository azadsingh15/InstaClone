## Insta Clone

This is a simple instagram clone in which a user can create his/her account. The user can also create posts follow other users and also comment on post of people we are following. 


#### How to run the project

1. Clone this project

```shell
git clone https://github.com/azadsingh15/InstaClone.git
```

2.  Move in the project directory `/InstaClone`

```shell
cd InstaClone
```

3. Set up your mongodb and paste the `MONGOURI` in `/InstaClone/config/dev.js`

4. Run the following command in `/InstaClone` and `/InstaClone/client` to install all the dependencies

```shell
npm i
```

5. Run the following command in `/InstaClone` and `/InstaClone/client` to start the client side and server side

```shell
npm start
```
